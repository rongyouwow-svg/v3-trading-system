# 🔧 页面空白问题修复指南

**问题时间**: 2026-03-13 22:45

---

## 🔍 问题分析

### 现象
- 页面能正常加载
- 左侧导航显示正常
- 右侧内容区域空白
- 右上角显示登录状态（admin）

### 原因
**浏览器缓存了旧版本的 JavaScript 代码**

旧版本中：
- 导航链接指向独立页面（`/dashboard/strategies.html`）
- 没有单页应用的 section 切换逻辑

新版本中：
- 导航链接使用 `data-section` 切换
- 需要加载对应的内容区域

---

## ✅ 解决方案

### 方案 1: 强制刷新浏览器 ⭐ 推荐

**Windows/Linux**:
```
Ctrl + Shift + R
```
或
```
Ctrl + F5
```

**Mac**:
```
Cmd + Shift + R
```

---

### 方案 2: 清除浏览器缓存

**Firefox**:
1. 按 `F12` 打开开发者工具
2. 右键点击刷新按钮
3. 选择"强制刷新"

**Chrome**:
1. 按 `F12` 打开开发者工具
2. 长按刷新按钮
3. 选择"清空缓存并硬性重新加载"

---

### 方案 3: 添加 URL 参数强制刷新

访问时添加版本号参数：
```
http://147.139.213.181:3000/dashboard/index.html?v=202603132245
```

---

### 方案 4: 检查浏览器控制台

1. 按 `F12` 打开开发者工具
2. 切换到 **Console（控制台）** 标签
3. 查看是否有 JavaScript 错误
4. 截图发送给我
