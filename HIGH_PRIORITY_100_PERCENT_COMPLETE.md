## 🎯 核心功能演示

### API 配置管理

**操作流程**:
1. 点击"➕ 添加 API"
2. 填写配置 ID、名称、环境、API Key、Secret Key
3. 点击"💾 保存"
4. 切换开关启用/禁用
5. 点击"⭐ 设为默认"设置默认 API
6. 点击"⚙️ 编辑"修改配置
7. 点击"🗑️ 删除"删除配置

---

### 插件配置

**Telegram 配置流程**:
1. 进入"插件管理"页面
2. 找到"Telegram 通知"插件
3. 点击"⚙️ 配置"
4. 输入 Bot Token（从 @BotFather 获取）
5. 输入 Chat ID（从 @userinfobot 获取）
6. 点击"💾 保存"
7. 点击"🔌 测试发送"验证配置

**钉钉配置流程**:
1. 进入"插件管理"页面
2. 找到"钉钉通知"插件
3. 点击"⚙️ 配置"
4. 输入 Webhook URL（从钉钉机器人设置获取）
5. 输入 Secret（加签密钥，可选）
6. 点击"💾 保存"
7. 点击"🔌 测试发送"验证配置
