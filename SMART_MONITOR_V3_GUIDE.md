## 📊 监测维度

### L1: 进程级监测

**动态获取预期策略**:
```python
def get_expected_strategies():
    # 从 Supervisor 配置读取
    # 不硬编码数量
    return ['quant-strategy-eth', ...]
```

**弹性判断**:
```python
if actual < expected:
    告警（进程不足）
elif actual > expected * 2:
    告警（进程过多）
else:
    正常（允许弹性）
```

---

### L2: 业务级监测

**币种维度检查**:
```python
# 每个币种只能有 1 个策略
for symbol, strat_list in strategy_symbols.items():
    if len(strat_list) > 1:
        告警(f"{symbol}: {len(strat_list)}个策略同时运行")
```

**持仓 - 止损单匹配**:
```python
# 1 持仓 = 1 止损单
for symbol in position_symbols:
    if stop_count != 1:
        告警()
```

---

### L3: 生命周期监测

**注册中心**:
- 策略启动 → 注册
- 策略停止 → 注销
- 动态更新保护清单

---

### L4: 自动修复

| 问题 | 修复方式 |
|------|---------|
| 止损单重复 | 自动清理 |
| 进程不足 | 告警 |
| 进程过多 | 告警 |
| 重复策略 | 告警 |
| 孤儿止损单 | 告警 |

## ✅ 支持的场景

| 场景 | 结果 |
|------|------|
| 开 A,B,C | ✅ 正常 |
| 关 A | ✅ 正常（不告警） |
| 关 A → 开 D | ✅ 正常 |
| ETH-A → ETH-B | ⚠️ 告警 + 清理 |
| 进程重复 | ⚠️ 告警 |
| 止损单重复 | 🗑️ 自动清理 |
