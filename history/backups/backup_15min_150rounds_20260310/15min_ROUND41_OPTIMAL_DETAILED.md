## 🔍 信号详解

### RSI(7) 计算
```python
RSI = 100 - (100 / (1 + RS))
RS = 过去 7 根 K 线平均涨幅 / 过去 7 根 K 线平均跌幅

多头信号：RSI < 18
空头信号：RSI > 82
```

### 布林带计算
```python
中轨 = SMA(收盘价，20)
标准差 = StdDev(收盘价，20)
上轨 = 中轨 + 2 × 标准差
下轨 = 中轨 - 2 × 标准差

多头信号：收盘价 < 下轨
空头信号：收盘价 > 上轨
```

### 成交量确认
```python
均量 = SMA(成交量，20)
成交量比率 = 当前成交量 / 均量

多头/空头信号：成交量比率 > 3.0
```
