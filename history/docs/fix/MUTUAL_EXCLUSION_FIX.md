### 问题描述
**截图显示：** 同一个币种（LINKUSDT）启动了 2 个相同的策略

**原因：** 缺少互斥检查机制

---

## 🛡️ 互斥机制实现

### 核心逻辑

```python
def start_strategy(self, symbol: str, ...):
    # 互斥检查：同一交易对只能有一个活跃策略
    for s in self.strategies:
        if s['symbol'] == symbol and s['status'] == '运行中':
            return {
                'success': False,
                'error': f'{symbol} 已有活跃策略，不能重复启动',
                'existing_strategy': s
            }
    
    # 启动新策略
    ...
```

### 互斥规则

**规则：** 每个交易对（symbol）同时只能有 **1 个活跃策略**

**检查条件：**
- ✅ 同一 symbol
- ✅ 状态为"运行中"
- ✅ 任意方向（LONG/SHORT 都互斥）

**触发结果：**
- ❌ 拒绝启动新策略
- ⚠️ 返回错误信息
- 📱 显示现有策略详情

## 💡 最佳实践

### 启动策略前
1. **检查现有策略** - 查看是否有同币种策略
2. **确认方向** - LONG 还是 SHORT
3. **评估风险** - 仓位是否合理

### 需要切换策略时
1. **先停止旧策略** - 平仓现有策略
2. **等待确认** - 确认已停止
3. **启动新策略** - 开新仓
