# 🦞 龙虾王量化交易方案说明书 v2.0

## 一、交易时间框架体系

### 核心原则：多周期共振

| 周期 | 作用 | 决策权重 |
|------|------|----------|
| **日线 (1D)** | 定方向、判趋势 | 40% |
| **4小时 (4H)** | 找结构、选时机 | 35% |
| **1小时 (1H)** | 精细进出、止损 | 25% |

### 进场标准
```
做多条件：
1. 日线趋势向上 (EMA20 > EMA50)
2. 4H回踩关键支撑位
3. 1H出现反转信号 (RSI<40 + MACD金叉)

做空条件：
1. 日线趋势向下 (EMA20 < EMA50)
2. 4H反弹至关键阻力位
3. 1H出现反转信号 (RSI>60 + MACD死叉)
```

## 二、支撑压力位计算体系

### 计算方法 (日线级别+)

**1. 经典枢轴点 (Pivot Points)**
```
Pivot = (前日高 + 前日低 + 前日收) / 3
R1 = 2*Pivot - 前日低
R2 = Pivot + (前日高 - 前日低)
S1 = 2*Pivot - 前日高
S2 = Pivot - (前日高 - 前日低)
```

**2. 斐波那契回撤**
- 取最近一波上涨/下跌的波段
- 关键位：0.236, 0.382, 0.5, 0.618, 0.786

**3. 前高前低聚合**
- 扫描30-60天内的局部极值点
- 聚类相近价位形成支撑阻力区

### 输出格式
```json
{
  "pivot": 1950.50,
  "resistance_levels": [2100.00, 2050.50, 2000.00],
  "support_levels": [1900.00, 1850.25, 1800.00],
  "nearest_resistance": 2000.00,
  "nearest_support": 1900.00,
  "key_highs": [2200.00, 2150.00],
  "key_lows": [1800.00, 1750.00]
}
```
