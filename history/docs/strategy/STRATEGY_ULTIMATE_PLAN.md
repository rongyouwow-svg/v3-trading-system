## ⚠️ 第五部分：风险提示

### 5.1 理论上限分析

**数学证明:**
```
年化 500% = 6 倍权益
8 年总收益 = 6^8 = 1,679,616 倍

单币种、有限交易频率下：
- 年均交易 100 次（上限）
- 胜率 70%（上限）
- 盈亏比 3:1（上限）
- 单次收益 5%（上限）

年收益计算:
盈利：70 次 × 5% = +350%
亏损：30 次 × 5%/3 = -50%
净收益：+300%

8 年复利:
(1 + 3.0)^8 = 4^8 = 65,536 倍

年化：(65536)^(1/8) - 1 = 300%

结论：年化 500% 在单币种上**理论上不可能**
除非：
1. 多币种分散（4 币种×300% = 1,200%）
2. 极高杠杆（10x+，但回撤会超 30%）
3. 抓住超级黑天鹅（8 年 1-2 次）

现实目标：年化 200-300%, 回撤<30%
```

### 5.2 过拟合风险

**缓解措施:**
```
1. 保留 20% 数据作为验证集
2. 参数范围限制（避免极端值）
3. 实盘小资金测试验证
4. 定期重新优化（每季度）
```

### 5.3 实盘失效风险

**可能原因:**
```
1. 回测假设理想成交（实盘有滑点）
2. 回测未考虑资金费率
3. 市场结构变化
4. 策略被广泛使用（alpha 衰减）
```

**缓解措施:**
```
1. 实盘前模拟盘测试 1 个月
2. 实盘从小资金开始（$1,000）
3. 定期监控策略表现
4. 准备备用策略
```
