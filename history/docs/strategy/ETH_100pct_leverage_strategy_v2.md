## 🔧 动态杠杆调整

### 根据波动率调整

```python
volatility = ATR(14) / close

if volatility > 0.05:  # 高波动
    leverage = 2  # 降至 2 倍
elif volatility < 0.02:  # 低波动
    leverage = 4  # 升至 4 倍
else:  # 正常波动
    leverage = 3  # 维持 3 倍
```

### 根据连胜/连败调整

```python
if consecutive_wins >= 3:
    leverage = min(leverage + 1, 4)  # 加仓
elif consecutive_losses >= 2:
    leverage = max(leverage - 1, 2)  # 减仓
```

### 根据回撤调整

```python
if total_drawdown > 0.25:  # 回撤>25%
    leverage = 2  # 降至 2 倍
if total_drawdown > 0.35:  # 回撤>35%
    leverage = 1  # 降至 1 倍
if total_drawdown > 0.45:  # 回撤>45%
    pause_trading()  # 暂停交易
```
