## 🎯 交易闭环流程

### 完整流程图

```
1. 策略启动
   ↓
2. 实时监控价格（每 5 秒）
   ↓
3. 计算盈亏
   ├─ 更新策略卡片显示
   ├─ 更新运行时间
   └─ 更新盈亏统计
   ↓
4. 生成交易信号（波动>1%）
   ↓
5. 执行交易
   ├─ 创建交易记录（pending）
   ├─ 调用币安 API 下单
   ├─ 等待成交确认
   └─ 更新交易记录（executed）
   ↓
6. 关闭策略
   ↓
7. 保存交易记录
   ↓
8. 更新账户余额
   ↓
9. 在交易记录页面显示
```

## 📝 API 端点

### 策略相关

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/strategy/list` | GET | 获取策略列表（含详细信息） |
| `/api/strategy/start` | POST | 启动策略 |
| `/api/strategy/stop` | POST | 停止策略 |
| `/api/strategy/signals` | POST | 生成交易信号 |
| `/api/strategy/execute` | POST | 执行交易信号 |

### 交易记录相关

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/trades` | GET | 获取交易记录列表 |
| `/api/trades/stats` | GET | 获取交易统计 |
| `/api/trades/<id>` | GET | 获取单条交易记录 |
