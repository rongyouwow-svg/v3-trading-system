## 📊 1. 单参数敏感度测试

### 1.1 Lookback 周期（突破周期）


### 1.2 Volume Multiplier（成交量放大倍数）

```
最佳值：1.2
稳健区间：(1.0, 1.2)
最佳收益：22.0%
```

**分析**: 成交量确认是过滤假突破的关键。过低（1.0-1.2）无法有效过滤，过高（2.5-3.0）会错过机会。

### 1.3 Stop Loss Percentage（止损百分比）

```
最佳值：0.01
稳健区间：(0.01, 0.02)
最佳收益：19.0%
```

**分析**: 止损设置需要平衡：太紧容易被扫止损，太松会增加单笔亏损。

### 1.4 Take Profit Ratio（盈亏比）

```
最佳值：2.0
稳健区间：(2.0, 2.0)
最佳收益：17.3%
```

**分析**: 高盈亏比（3.0+）需要更高的趋势延续性，低盈亏比（1.5-2.0）更依赖胜率。
