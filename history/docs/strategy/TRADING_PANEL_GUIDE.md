# 🦞 交易设置面板使用指南

## 📋 功能恢复

已完整恢复原有的交易设置面板，包括：

### 设置项

1. **账户类型** - 测试网/实盘切换
2. **杠杆倍数** - 1-5x 可选
3. **选择币种** - BTC/ETH/LINK/UNI
4. **选择策略** - v23/v24/v25
5. **交易金额** - USDT 数量

### 操作按钮

- ▶️ **开始交易** - 启动策略
- ⏹️ **停止交易** - 停止当前策略
- 🛑 **停止全部** - 停止所有策略

---

## 🚀 使用步骤

### 测试网交易

1. 访问：`http://147.139.213.181:8080/quant/pages/testnet.html`
2. 设置参数：
   - 账户类型：测试网
   - 杠杆：1x（新手建议）
   - 币种：BTCUSDT
   - 策略：v23 高频
   - 金额：100 USDT
3. 点击"开始交易"
4. 查看持仓和盈亏

### 模拟盘交易

1. 访问：`http://147.139.213.181:8080/quant/pages/sim.html`
2. 设置参数：
   - 杠杆：1x
   - 币种：ETHUSDT
   - 策略：v23 高频
   - 金额：10 USDT（模拟盘初始$100）
3. 点击"开始交易"
4. 查看持仓变化

### 实盘交易

⚠️ **谨慎操作！**

1. 先在 API 配置页面添加实盘 API Key
2. 访问：`http://147.139.213.181:8080/quant/pages/real.html`
3. 设置参数：
   - 账户类型：实盘
   - 杠杆：1x（强烈建议）
   - 币种：BTCUSDT
   - 策略：v23 高频
   - 金额：**小额测试**
4. 点击"开始交易"
5. 密切监控

## 📊 策略说明

### v23 高频策略
- 适合震荡行情
- 交易频率高
- 单次盈利小

### v24 多周期
- 多时间周期分析
- 适合趋势行情
- 交易频率中
- 风险：中低

### v25 专业版
- 综合多种指标
- 适合各种行情
- 交易频率低
- 风险：低

## ⚠️ 风险提示

1. **杠杆风险** - 高杠杆可能爆仓
2. **策略风险** - 策略可能失效
3. **市场风险** - 极端行情可能亏损
4. **技术风险** - 网络/系统可能故障

**建议：**
- 只用闲钱
- 设置止损
- 不要 All-in
- 持续学习

## 🔧 故障排查

### 问题 1：点击开始交易无反应
**解决：**
1. 打开浏览器开发者工具（F12）
2. 查看 Console 错误
3. 检查网络连接
4. 刷新页面重试

### 问题 2：显示"无法获取价格"
**解决：**
1. 检查 API 服务状态
2. 刷新页面
3. 清除浏览器缓存

### 问题 3：策略启动失败
**解决：**
1. 检查账户余额
2. 降低交易金额
3. 查看错误提示
