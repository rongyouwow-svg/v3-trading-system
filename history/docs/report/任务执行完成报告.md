**修改文件：** `gateway.py`

**实现内容：**
```python
# 取消所有 Algo 订单（止损单/止盈单）
algo_orders = client.get_algo_orders(symbol=symbol, limit=10)
cancelled_count = 0
if algo_orders.get('success'):
    for order in algo_orders.get('orders', []):
        if order.get('algoStatus') == 'NEW':
            cancel_result = client.cancel_algo_order(symbol, order['algoId'])
            if cancel_result.get('success'):
                cancelled_count += 1
```

**效果：**
- ✅ 关闭策略时自动取消所有止损单
- ✅ 防止止损单继续挂单
