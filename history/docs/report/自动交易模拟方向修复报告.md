# 🦞 自动交易模拟方向问题修复报告

**修复时间：** 2026-03-10 12:23  
**问题：** 自动交易模拟策略显示空单（SHORT），应该是多单（LONG）  
**状态：** ✅ 已修复  

---

## 🔍 问题根源

**前端代码硬编码了方向：**
```javascript
// pages/testnet.html 第 625 行（修复前）
body: JSON.stringify({
    symbol,
    strategy,
    side: 'short',  // ❌ 硬编码为 short
    ...
})
```

**问题：**
- 无论选择什么策略，前端都传递 `side: 'short'`
- 后端修复了（auto_sim 默认 long），但前端覆盖了
- 导致 auto_sim 策略仍然是空单

---

## ✅ 修复内容

### 1. 前端方向修复

**修改 `pages/testnet.html`：**
```javascript
// 修复后
const side = strategy === 'auto_sim' ? 'long' : 'short';
const actionText = side === 'long' ? 'LONG' : 'SHORT';

body: JSON.stringify({
    symbol,
    strategy,
    side: side,  // ✅ 根据策略决定
    ...
})

// 提示文字也改为动态
alert(`✅ 交易已启动\n${symbol} ${actionText} @ $${price}...`);
```

### 2. 持仓显示增强

**修改 `pages/testnet.html` 持仓渲染：**
```javascript
// 新增显示:
杠杆：${p.leverage || 1}x
仓位：$${p.position_value?.toFixed(2) || '0.00'}
保证金：$${p.margin?.toFixed(2) || '0.00'}
爆仓价：$${p.liquidation_price.toFixed(2)}
```
