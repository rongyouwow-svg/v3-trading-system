## 阶段 1：识别主力活动片段（2 小时）

**目标：** 从 8.5 年数据中找出所有主力活动痕迹

**任务清单：**
1. 定义主力活动识别标准（成交量>3 倍 + 涨跌幅>2%）
2. 扫描 1 分钟全量数据，标记主力活动时刻
3. 提取每次活动前后 50 根 K 线数据
4. 统计主力活动总次数（预计 500-1000 次）

**产出：**
- 主力活动时间列表（CSV）
- 每次活动的前后 K 线片段

## 阶段 2：提取活动特征（2 小时）

**目标：** 为每次主力活动建立特征档案

**任务清单：**
1. 提取时间特征（小时/周几/月份）
2. 提取价格特征（RSI/布林位置/EMA 斜率）
3. 提取成交量特征（放量倍数/持续时间）
4. 提取 K 线特征（影线长度/实体大小）
5. 提取后续走势特征（最大盈利/最大回撤）

**产出：**
- 主力活动特征矩阵（1000 次×20 特征）
- 主力行为画像初稿

## 阶段 3：聚类找模式（2 小时）

4. 可视化聚类结果

**产出：**
- 3-5 个典型主力模式
- 每个模式的特征描述
- 聚类可视化图表

## 阶段 4：归纳主力习惯（2 小时）

**目标：** 总结主力操作规律

**任务清单：**
1. 分析每个模式的时间偏好（几点/周几活跃）
2. 分析每个模式的指标偏好（RSI/布林位置）
3. 分析每个模式的操作手法（建仓/洗盘/拉升/出货）

**产出：**
- ETH 主力行为画像报告
