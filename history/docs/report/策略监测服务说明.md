## ⚠️ 异常情况处理

### 情况 1：策略有记录但无持仓
```
可能原因:
- 下单失败
- 持仓被意外平仓

处理方法:
1. 检查交易记录确认下单结果
2. 如需重新下单，停止策略后重新启动
```

### 情况 2：数量不一致
```
可能原因:
- 精度处理问题
- 部分平仓

处理方法:
1. 检查精度设置
2. 确认是否有部分平仓操作
```

### 情况 3：方向不一致
```
可能原因:
- 策略方向记录错误
- 持仓方向判断错误

处理方法:
1. 检查策略引擎逻辑
2. 检查持仓 API 返回数据
```
