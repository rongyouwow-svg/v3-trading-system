```

- stopPrice: 触发价格
```

**问题：** 测试网可能不支持

### 方案 2：本地监控止损（备选）

**逻辑：**
```python
# 策略引擎定期检查价格
if current_price <= stop_price:  # 多单止损
```

- 不依赖币安止损单 API
