## 第一章：设计概述

### 1.1 设计理念

**核心设计原则：**
- **简洁直观** - 数据清晰可见，操作简便
- **科技风格** - 深色背景，荧光色强调
- **数据驱动** - 实时数据展示，状态即时反馈
- **用户友好** - 错误提示清晰，加载状态明确

### 1.2 设计风格

**风格定位：** 深色科技风

**视觉特征：**
```
背景：深蓝渐变色
强调：荧光绿色（#00ff88）
文字：白色为主，灰色为辅
边框：半透明白色
圆角：6-16px
阴影：轻微阴影或无阴影
```

### 1.3 适用场景

**主要场景：**
- 量化交易管理界面
- 账户资产管理
- 持仓监控
- 策略管理
- 交易记录查看

**不适用场景：**
- 移动端 App（需单独设计）
- 打印输出（深色背景不适合打印）
- 正式商务场景（风格过于科技化）

### 1.4 浏览器兼容性

**支持的浏览器：**
- ✅ Chrome 80+
- ✅ Firefox 75+
- ✅ Safari 13+
- ✅ Edge 80+

**不支持的浏览器：**
- ❌ Internet Explorer（已停止支持）

**特性要求：**
- ES6+ JavaScript
- CSS Grid 和 Flexbox
- CSS 变量（可选）

```
✅ .card-success
❌ .btnDanger

```

### 9.2 CSS 规范

**选择器规范：**
```css
/* ✅ 好的写法 */
.card { }
.card.success { }
.card > div { }

/* ❌ 不好的写法 */
div.card { }  /* 避免标签选择器 */
.card .card { }  /* 避免嵌套过深 */
```

**属性顺序：**
```css
.card {
    /* 定位 */
    position: relative;
    
    /* 布局 */
    display: flex;
    
    /* 尺寸 */
    width: 100%;
    
    /* 外观 */
    background: rgba(255, 255, 255, 0.05);
    
    /* 其他 */
    border-radius: 12px;
}
```

### 9.3 JavaScript 规范

**函数命名：**
```javascript
// ✅ 好的写法
async function loadAccount() { }
function formatDate(date) { }

// ❌ 不好的写法
async function load_account() { }
function formatdate(date) { }
```

**异步处理：**
```javascript
// ✅ 好的写法 - 使用 async/await
async function loadData() {
    try {
        const resp = await fetch(url);
        const data = await resp.json();
        return data;
    } catch (e) {
        console.error('加载失败:', e);
        throw e;
    }
}

// ❌ 不好的写法 - 回调地狱
function loadData(callback) {
    fetch(url).then(function(resp) {
        resp.json().then(function(data) {
            callback(data);
        });
    });
}
```

**错误处理：**
```javascript
// ✅ 好的写法 - 完整的错误处理
async function loadData() {
    try {
        const resp = await fetch(url);
        
        if (!resp.ok) {
            throw new Error(`HTTP ${resp.status}`);
        }
        
        const data = await resp.json();
        
        if (!data.success) {
            throw new Error(data.error || '请求失败');
        }
        
        return data;
    } catch (e) {
        console.error('加载失败:', e);
        throw e;
    }
}
```

```javascript
    for (let i = 0; i < maxRetries; i++) {
        try {
            const resp = await fetch(url);
            if (resp.ok) return await resp.json();
```

```javascript
// 缓存数据，避免频繁请求
let cachedData = null;
let cacheTime = 0;

async function getCachedData() {
    const now = Date.now();
    
    // 5 分钟内使用缓存
    if (cachedData && (now - cacheTime) < 300000) {
        return cachedData;
    }
    
    const data = await fetchData();
    cachedData = data;
    cacheTime = now;
    
    return data;
}
```

**防抖节流：**
```javascript
// 防抖 - 避免频繁刷新
function debounce(func, wait) {
    let timeout;
    return function(...args) {
        clearTimeout(timeout);
        timeout = setTimeout(() => func.apply(this, args), wait);
    };
}

// 使用示例
const refreshData = debounce(async () => {
    await loadData();
}, 1000);  // 1 秒内只执行一次
```
