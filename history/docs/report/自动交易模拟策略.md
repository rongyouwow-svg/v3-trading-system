# 🦞 自动交易模拟策略

**策略名称：** 自动交易模拟  
**创建时间：** 2026-03-10 10:21  
**用途：** 模拟盘自动交易测试  
**周期：** 1 分钟  

---

## 📊 策略逻辑

### 循环交易流程
```
第 1 分钟：开多 50%（使用 50% 交易金额）
   ↓
第 2 分钟：加仓 30%（使用 30% 交易金额）
   ↓
第 3 分钟：平多 100%（平仓全部持仓）
   ↓
循环回到第 1 分钟...
```

### 交易规则
```
开多（50%）:
├─ 动作：BUY
├─ 类型：OPEN
├─ 仓位：50% 交易金额
└─ 杠杆：前端设置

加仓（30%）:
├─ 动作：BUY
├─ 类型：ADD
├─ 仓位：30% 交易金额
└─ 杠杆：前端设置

平仓（100%）:
├─ 动作：SELL
├─ 类型：CLOSE
├─ 仓位：100% 持仓
└─ 盈亏：实时计算
```

## ⚙️ 策略配置

### 参数说明
| 参数 | 默认值 | 说明 | 来源 |
|------|--------|------|------|
| symbol | ETHUSDT | 交易对 | 可配置 |
| initial_capital | $10,000 | 初始资金 | 可配置 |
| leverage | 10x | 杠杆倍数 | **前端设置** |
| trade_amount | $1,000 | 每次交易金额 | **前端设置** |

### 前端控制参数
```
策略从前端读取:
├─ 杠杆倍数（leverage）
└─ 交易金额（trade_amount）

示例:
├─ 杠杆：5x / 10x / 20x
└─ 金额：$100 / $500 / $1000
```

## 🔧 使用方法

### 方法 1：独立运行（测试）
```bash
cd /home/admin/.openclaw/workspace/quant
python3 auto_trading_sim.py
```

输出：
- 生成 10 个模拟信号
- 保存到 `auto_trading_sim_signals.json`
- 显示策略状态

### 方法 2：集成到前端（实盘）
```python
from auto_trading_sim import AutoTradingSimulator

# 创建模拟器
simulator = AutoTradingSimulator(symbol="ETHUSDT", initial_capital=10000)

# 从前端获取参数
simulator.update_params(leverage=10, trade_amount=1000)

# 每分钟调用（获取当前价格）
signal = simulator.generate_signal(current_price=2000)

# 发送信号到前端
if signal:
    send_to_frontend(signal)
```

### 方法 3：定时任务（自动执行）
```bash
# 每分钟执行一次（使用 cron）
* * * * * cd /home/admin/.openclaw/workspace/quant && python3 auto_trading_sim.py
```
