## ✅ 核心设计确认

### 确认 1：仓位计算方式

**确认内容：**
```
✅ 限制保证金占用不超过 80%
✅ 不是限制总仓位价值
✅ 符合币安实际计算方式
```

**计算公式：**
```
可用保证金 = 账户余额 - 已占用保证金
需要保证金 = (数量 × 价格) / 杠杆
最大允许 = 可用保证金 × 0.80

如果需要保证金 > 最大允许：
  → 拒绝开单
```

**配置项：**
```json
{
    "risk_control": {
        "max_margin_usage_pct": 0.80
    }
}
```

## 📊 数据架构

### Data Feed（数据层）

**职责：**
- 获取最新价格
- 缓存价格数据
- 推送给 Kline Generator

**代码结构：**
```python
class DataFeed:
    def __init__(self, client):
        self.client = client
        self.prices = {}
    
    def get_price(self, symbol):
        """获取最新价格"""
        return self.client.get_spot_price(symbol)
    
    def get_all_prices(self, symbols):
        """获取多个币种价格"""
        return {s: self.get_price(s) for s in symbols}
```

**职责：**

## 🛡️ 风控设计

### 混合止损模式

**设计原则：**
```
1. 默认开启硬止损（5%）
2. 策略可声明自定义止损
3. 配置文件中可开关
```

**止损类型：**
| 类型 | 触发条件 | 执行者 | 优先级 |
|------|---------|--------|--------|
| 硬止损 | 开仓价±5% | 网关 | 低 |
| 策略止损 | 策略计算价格 | 策略 | 高 |

### 保证金占用检查

**代码逻辑：**
```python
def check_margin_usage(symbol, quantity, price, leverage):
    account = client.get_futures_account()
    available_margin = float(account['availableBalance'])
    required_margin = (quantity * price) / leverage
    max_allowed = available_margin * 0.80
    
    if required_margin > max_allowed:
        return False, "保证金占用超过 80%"
    
    return True, "通过"
```
