## 📊 策略逻辑

完整的三步交易流程：

```
T+0 秒  → 开仓 50%  (OPEN, BUY, 50%)
   ↓
等待 5 秒
   ↓
T+5 秒  → 加仓 30%  (ADD, BUY, 30%)
   ↓
等待 5 秒
   ↓
T+10 秒 → 平仓 100% (CLOSE, SELL, 100%)
```

## 🔧 故障排查

### 策略无法启动

**可能原因：**
- 策略 ID 错误
- 已有活跃策略

**解决方案：**
```bash
# 停止现有策略
curl -X POST "http://localhost:8080/api/strategies/stop" \
  -H "Content-Type: application/json" \
  -d '{"symbol": "ETHUSDT"}'

# 重新启动
```

### 交易记录不显示

**可能原因：**
- 交易尚未执行完成
- 前端缓存问题

**解决方案：**
1. 等待 15 秒（策略执行时间 10 秒）
2. 刷新交易记录
3. 清除浏览器缓存

### 策略执行失败

**查看日志：**
```bash
curl "http://localhost:8080/api/strategies/logs?symbol=ETHUSDT"
```

**常见错误：**
- 资金不足
- 仓位超限
- API 调用失败
