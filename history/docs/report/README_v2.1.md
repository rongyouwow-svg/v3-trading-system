# 🦞 币安量化交易系统 v2.1 - 测试网版

## 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                      前端层 (8080 端口)                       │
├─────────────────────────────────────────────────────────────┤
│  /quant/pages/testnet.html  → 测试网交易 (绿色，默认)         │
│  /quant/pages/real.html     → 实盘交易 (红色)                │
│  /quant/pages/api_config.html → API 配置管理 (紫色)          │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                    API 服务层 (5005 端口)                      │
├─────────────────────────────────────────────────────────────┤
│  /api/testnet/*             → 币安测试网 API                  │
│  /api/real/*                → 币安实盘 API                    │
│  /api/config/*              → 配置管理 API                    │
│  /api/strategy/*            → 策略管理 API                    │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                      数据层                                  │
├─────────────────────────────────────────────────────────────┤
│  测试网：https://testnet.binancefuture.com                  │
│  实盘网：https://fapi.binance.com                           │
│  默认测试网 API：已配置 (无需添加)                           │
└─────────────────────────────────────────────────────────────┘
```

## 🎯 测试网 vs 本地模拟 vs 实盘

| 特性 | 测试网 | 本地模拟 | 实盘 |
|------|--------|----------|------|
| 资金 | 虚拟 USDT | 虚拟 USDT | 真实资金 |
| API 环境 | ✅ 真实 | ❌ 模拟 | ✅ 真实 |
| 订单执行 | ✅ 真实 | ❌ 模拟 | ✅ 真实 |
| 速率限制 | ✅ 有 | ❌ 无 | ✅ 有 |
| 网络延迟 | ✅ 有 | ❌ 无 | ✅ 有 |
| 风险 | ✅ 无 | ✅ 无 | ❌ 有 |
| 代码修改 | ❌ 无需 | ❌ 需修改 | ❌ 无需 |

**结论：测试网是最优选择！**

## 💡 使用建议

### 新手流程

1. ✅ **测试网练习** (1-2 周)
   - 熟悉 API 调用
   - 测试交易逻辑
   - 验证策略

2. ✅ **小额实盘** (1 个月)
   - 投入少量资金
   - 验证实盘环境
   - 监控表现

3. ✅ **正常运营**
   - 逐步增加资金
   - 优化策略
   - 持续监控

### 安全提示

⚠️ **实盘注意事项：**

1. **API Key 安全**
   - 不要分享
   - 禁用提现权限
   - 设置 IP 白名单
   - 定期轮换

2. **资金管理**
   - 只用闲钱
   - 设置止损
   - 不要 All-in
   - 定期出金

3. **策略风险**
   - 先测试后实盘
   - 监控回撤
   - 设置最大亏损
   - 不要过度优化

## 🔧 故障排查

### 测试网无法连接

```bash
# 检查 API 服务
curl http://localhost:5005/api/testnet/account

# 查看日志
tail -f /tmp/api_v2.log
```

### 订单失败

- 检查 API Key 权限
- 检查速率限制
- 检查账户余额
- 查看错误信息

### 价格不更新

- 检查网络连接
- 检查 API 限流
- 刷新页面
