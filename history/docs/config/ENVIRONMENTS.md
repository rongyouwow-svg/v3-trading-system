## 数据流对比

### 测试网数据流
```
前端 → /api/testnet/* → BinanceClient(testnet=True) 
     → https://testnet.binancefuture.com 
     → 返回测试网数据
```

### 模拟盘数据流
```
前端 → /api/sim/* → SimulatedAccount 
     → /tmp/sim_state_v6.json 
     → 返回本地数据
```

### 实盘数据流
```
前端 → /api/real/* → BinanceClient(testnet=False) 
     → https://fapi.binance.com 
     → 返回实盘数据
```

## 常见错误

### 测试网错误

**错误 1：未配置 API Key**
```
解决：系统已默认配置，无需手动添加
```

**错误 2：速率限制**
```
解决：降低请求频率，添加延时
```

### 模拟盘错误

**错误 1：资金不足**
```
解决：点击"重置账户"恢复初始资金
```

**错误 2：持仓不存在**
```
解决：先开仓再平仓
```

### 实盘错误

**错误 1：API Key 无效**
```
解决：检查 API Key 是否正确，权限是否开启
```

**错误 2：签名错误**
```
解决：检查系统时间是否同步
```
