## 🔧 配置步骤

### 1️⃣ 创建 Telegram Bot

1. 在 Telegram 搜索 **@BotFather**
2. 发送 `/newbot`
3. 按提示设置：
   - 机器人名称：`My Quant Bot`
   - 用户名：`my_quant_bot`（必须以 bot 结尾）
4. 复制 **Bot Token**（格式：`1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`）

### 2️⃣ 获取 Chat ID

1. 在 Telegram 搜索你的机器人并启动
2. 发送任意消息（如 `/start`）
3. 访问：
   ```
   https://api.telegram.org/bot<BOT_TOKEN>/getUpdates
   ```
4. 在返回结果中找到：
   ```json
   {
     "result": [
       {
         "message": {
           "chat": {
             "id": 123456789,
             ...
           }
         }
       }
     ]
   }
   ```
5. 复制 `id` 值（数字，如 `123456789`）

### 3️⃣ 配置到系统

1. 访问配置页面：
   ```
   http://147.139.213.181:8080/pages/telegram_config.html
   ```

2. 填写：
   - **Bot Token:** `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`
   - **Chat ID:** `123456789`

3. 点击 **"保存配置"**

4. 点击 **"测试通知"** 验证
