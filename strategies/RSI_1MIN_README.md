# 🦞 RSI 1 分钟反转策略

## 策略 ID
- `rsi1minreversal` (推荐)
- `rsi_1min_reversal` (兼容别名)

## 策略逻辑

### 开仓条件
- **RSI > 50** → 开空单

### 平仓条件
- **RSI < 50** → 平空单

### 止盈/止损
- **止盈**: 0.5%
- **止损**: 0.5%

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| rsi_period | 14 | RSI 计算周期 |
| rsi_threshold | 50 | RSI 开仓阈值 |
| take_profit_pct | 0.5% | 止盈百分比 |
| stop_loss_pct | 0.5% | 止损百分比 |

## 信号频率

根据实测（2026-03-13 ETHUSDT 1 分钟数据）：
- **约每 12.5 分钟** 产生 1 个开仓信号
- 3.3 小时内产生 16 次开仓信号
- 信号频率较高，适合测试执行能力

## 回测表现

测试时间：200 根 1 分钟 K 线（约 3.3 小时）
- 总收益率：-2.32%
- 总交易次数：32 次
- 止盈触发：2 次
- 止损触发：6 次

⚠️ **注意**: 此策略设计用于测试信号频率和执行能力，不是盈利策略

## 使用方法

### 通过网关 API 启动
```bash
POST /api/strategy/start
{
  "strategy_id": "rsi1minreversal",
  "symbol": "ETHUSDT",
  "leverage": 1,
  "amount": 100
}
```

### 通过策略列表选择
1. 访问网关导航页面
2. 选择"策略交易"
3. 选择"RSI 1 分钟反转策略"
4. 设置参数并启动
