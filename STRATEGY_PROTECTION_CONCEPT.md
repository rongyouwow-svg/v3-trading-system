# 🛡️ 策略保护系统正确理念

**版本**: v3.0 (用户纠正版)  
**创建时间**: 2026-03-19 18:20  
**核心原则**: 执行端自主 + 自动发现 + 动态保护

---

## ❌ 错误理解（我之前）

```
❌ Supervisor 配置决定运行哪些策略
❌ 监控系统保护配置的所有策略
❌ 添加/删除策略需要修改配置
❌ 保护清单 = 配置文件
```

---

## ✅ 正确理解（用户指导）

```
✅ 执行端（策略脚本）自己决定运行哪些
✅ 监控系统实时发现正在运行的策略
✅ 策略启动 → 自动加入保护清单
✅ 策略关闭 → 自动从保护清单剔除
✅ 保护清单 = 当前运行的策略
```

## 🔄 完整流程

### 策略启动流程

```
1. 策略脚本启动
   ↓
2. 初始化策略对象
   ↓
3. 调用 register_strategy(symbol)
   ↓
4. 注册中心记录：{symbol: {pid, start_time, ...}}
   ↓
5. 监控系统发现新策略
   ↓
6. 加入保护清单
   ↓
7. 开始监控和修复
```

---

### 策略停止流程

```
1. 策略脚本准备停止
   ↓
2. 调用 unregister_strategy(symbol)
   ↓
3. 注册中心删除记录
   ↓
4. 监控系统发现策略消失
   ↓
5. 从保护清单剔除
   ↓
6. 停止监控
```

---

### 异常处理流程

```
1. 监控系统检查注册中心
   ↓
2. 获取保护清单（运行中的策略）
   ↓
3. 检查每个策略状态
   ↓
4. 发现异常 → 自动修复
   ↓
5. 策略崩溃 → Supervisor 重启
   ↓
6. 重启后自动注册 → 继续保护
```

## 🔧 需要修复的问题

### 1. 注册中心 API 未实现

**问题**: `/api/strategies` 返回空数组

**修复**: 实现真实的注册中心 API

```python
# 策略注册中心
active_strategies = {}

@app.post("/api/strategy/register")
def register(symbol: str, pid: int):
    active_strategies[symbol] = {
        'pid': pid,
        'start_time': datetime.now(),
        'status': 'running'
    }

@app.post("/api/strategy/unregister")
def unregister(symbol: str):
    if symbol in active_strategies:
        del active_strategies[symbol]

@app.get("/api/strategies")
def get_strategies():
    return {
        'strategies': list(active_strategies.values()),
        'count': len(active_strategies)
    }
```

### 2. 监控系统基于固定列表

**问题**: 监控硬编码的策略列表

```python
# ❌ 错误
for strategy in ["eth", "link", "avax"]:
    check_strategy(strategy)

# ✅ 正确
strategies = get_active_strategies()
for strategy in strategies:
    check_strategy(strategy)
```

### 3. 保护清单不动态

**问题**: 保护清单是静态配置的

**修复**: 保护清单 = 注册中心的策略列表

```python
# 每次检查前获取最新保护清单
protection_list = get_active_strategies()

# 只保护清单内的策略
for strategy in protection_list:
    protect(strategy)
```

## 🎯 核心原则

1. **执行端自主** - 策略自己决定运行与否
2. **自动发现** - 监控系统自动发现新策略
3. **动态保护** - 保护清单实时更新
4. **注销即移除** - 策略停止后不再保护

---

**保护系统应该像保镖：**
- 老板出现 → 立即保护
- 老板离开 → 停止保护
- 老板换地方 → 跟随保护
