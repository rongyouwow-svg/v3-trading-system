## 🔍 复查内容

### 每次复查必须检查

1. **止损单数量** - 是否还在增加
2. **策略日志** - 是否有重复创建日志
3. **策略状态** - 是否正常运行
4. **持仓状态** - 是否正常

### 检查命令

```bash
# 1. 检查止损单数量
curl -s http://localhost:3000/api/binance/stop-loss | \
  python3 -c "import sys,json; d=json.load(sys.stdin); print(f'止损单：{len(d.get(\"orders\",[]))}个')"

# 2. 检查策略日志
tail -20 /root/.openclaw/workspace/quant/v3-architecture/logs/supervisor_eth_out.log | \
  grep -E "止损|create_stop"

# 3. 检查策略状态
supervisorctl status

# 4. 检查持仓
curl -s http://localhost:3000/api/binance/positions | \
  python3 -c "import sys,json; d=json.load(sys.stdin); print(f'持仓：{len(d.get(\"positions\",[]))}个')"
```
