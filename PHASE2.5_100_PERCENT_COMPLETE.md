## 📝 关键发现

### 止损单 API 端点
**正确端点**: `POST /fapi/v1/algoOrder`

**关键参数**:
```python
params = {
    "symbol": "ETHUSDT",
    "side": "SELL",
    "algoType": "CONDITIONAL",  # 只支持 CONDITIONAL
    "type": "STOP_MARKET",
    "triggerPrice": "2000",  # 使用 triggerPrice 不是 stopPrice
    "quantity": "0.01",
    "reduceOnly": "false"
}
```

**返回参数**:
- `orderId`: 订单 ID
- `orderStatus`: 订单状态
- `algoType`: CONDITIONAL
