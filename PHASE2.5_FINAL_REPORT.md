## 🔧 解决方案

### 需要配置 API Key 权限

请访问：https://demo-fapi.binance.com/fapi/v1/apiKey

**需要的权限**:
1. ✅ **Enable Reading** - 读取权限
2. ✅ **Enable Futures** - Futures 交易权限
3. ❌ **Disable Withdrawals** - 禁用提现（安全建议）

**IP 白名单**:
- 如果设置了 IP 限制，请添加当前服务器 IP
- 或暂时移除 IP 限制用于测试

**获取当前服务器 IP**:
```bash
curl ifconfig.me
```
